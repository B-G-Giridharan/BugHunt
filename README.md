# 🐞 Bug Hunt – Debugging Challenge

## 📌 Overview
Bug Hunt is a competitive coding event focused on real-time debugging. Participants are challenged to identify and fix **syntax**, **runtime**, and **logical errors** under strict constraints, testing both speed and accuracy.
This event was conducted as part of an international symposium **Zyphoria'26**.

## 🧠 Challenge Structure

### 🔹 Round 1
- 15 Questions 30 Minutes
- Focus: Syntax Errors & Runtime Errors
- Includes batch-based debugging tasks

### 🔹 Round 2
- 5 Questions 10 Minutes
- Focus: Logical Errors
- Advanced debugging scenarios
- Includes batch-based debugging tasks

---

## 👥 Participation Rules
- Team-based (2 members per team)
- No AI tools or internet access allowed
- Only Python IDE permitted
